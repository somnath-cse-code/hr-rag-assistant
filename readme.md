# HR Handbook Assistant

AI-powered HR chatbot using RAG architecture.

## Features

- Upload HR PDFs
- Semantic search
- AI question answering
- Local LLM using Ollama
- Chat history
- Source chunk display

## Tech Stack

- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Ollama
- Phi3

## Run Project

First run:

```bash
ollama run phi3
